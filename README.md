# Cosevawriter
Program to overwrite and read .csv files

## Installation
### Clone repository
```
git clone --recursive https://github.com/manucabral/Cosevawriter.git
cd Cosevawriter
```

### Run
```sh
g++ src/main.cpp -o coseva
coseva [filename] [write/read]
```
> Or you can also start the **Makefile**
